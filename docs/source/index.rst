Welcome to the documentation for Senko, an entertainment bot for Discord, themed
around Sewayaki Kitsune no Senko-san. Senko is written in `Python 3.9`_  and
uses the `discord.py`_ library. 

This documentation aims to cover the API of Senko's systems and other related
topics, such as installation guides, data formats or the various tools used to
create the assets used by Senko.

.. _Python 3.9: https://www.python.org/
.. _discord.py: https://github.com/Rapptz/discord.py
